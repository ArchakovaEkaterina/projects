В файле "Dock.docx" расписано задание к данной субд, а также описание предметной области и концептуальная информационная модель
Интерфейс главного меню включает в себя 3 кнопки: 
1.Бронирование билетов. Пользователь должен ввести данные, по которым будет производиться поиск рейсов, выбрать класс места и номер места, после ввести свои данные.
Результат - чек забронированных билетов
2. Поиск билетов пассажира. Производится по введенным данным пассажира(ФИО и паспорт)
Результат - отчет со всеми билетами пассажира
3. Поиск проданных билетов рейса. По введенным данным рейса ищутся все забронированные билеты.
